# 数据字段与抓取完善说明

版本：v0.1  
日期：2026-06-27

## 1. 本轮参考文档学习结论

参考文件：`2025～2026线下行业峰会、展会表.xlsx`

表格不是普通展会清单，而是已经带有 LHB 业务判断视角。可复用字段包括：

1. 基础字段：活动时间、活动名称、行业/领域、城市及场地、活动形式、主办方、详情网页。
2. 商务字段：建议参会BU、门票/获票、参会形式、参会费用。
3. 客户字段：与会人员公司类型、代表性企业/品牌、与会人员职级岗位。
4. 价值字段：规模、优势、备注。
5. 媒体字段：行业媒体/机构名称、主办/协办活动、媒体类型、粉丝量、平均阅读量。

这些字段对 LHB 的核心价值不是“信息更全”，而是帮助判断：

1. 是否值得去。
2. 应该谁去。
3. 去了找谁。
4. 用什么产品切。
5. 是否值得做头部专题曝光。
6. 是否能沉淀公众号/媒体/主办方持续抓取源。

## 2. 网页字段升级方向

当前页面已经新增或补齐以下字段：

1. 图片：列表、头部专题、详情均展示。
2. 来源类型：聚展网、活动行、官网/公开网页、公众号/行业媒体、本地参考表、待核验。
3. 可信度：高、中、低。
4. 规模与票务：展览面积、展商数、观众数、门票/会刊等。
5. 建议参会方式：参观、购买会刊、预约拜访、历史归档复盘等。
6. 与会人群：品牌方、渠道商、服务商、媒体、商场负责人等。
7. 与会职级：董事长、CEO、市场负责人、渠道负责人、采购负责人等。
8. 代表品牌/目标客户：从参考表和公开网页抽取。
9. 建议参会BU：BD、体验设计、内容专题、数智化选址等。
10. 公众号/媒体线索：后续抓取的重要入口。
11. LHB切入角度：用业务语言说明为什么这场活动值得跟进。

1. 上海
2. 杭州
3. 广州
4. 深圳
5. 北京

2. 香氛

## 4. 来源策略

### 4.1 展会聚合网站

可抓取来源包括：

1. 聚展网 `jufair.com`
2. 活动行 `huodongxing.com`
3. 展会官网
4. 主办方官网
5. 场馆活动日历

优先抓取字段：

1. 活动名称
2. 时间
3. 城市/场馆
4. 主办方
5. 展览面积、展商数、观众数
6. 门票/会刊
7. 展品范围
8. 往届品牌或代表企业
9. 官网链接

### 4.2 活动行

活动行是非常重要的补充来源，尤其适合补聚展网覆盖不足的活动类型：

1. 行业峰会
2. 沙龙/闭门会
3. 品牌发布会
4. 招商会/平台招商
5. 商业地产活动
6. 线下体验活动
7. 付费课程/工作坊

活动行抓取时要特别关注：

1. 活动城市是否在上海、杭州、广州、深圳、北京。
2. 活动是否对应 LHB 高匹配行业，而不是泛创业、泛职场、泛培训。
3. 主办方是否是行业媒体、协会、平台、品牌、商业地产或展会主办方。
4. 是否能看到嘉宾、议程、参会人群、票价、报名人数、活动地点。
5. 是否适合形成BD线索、内容专题、赞助曝光或活动合作。

活动行优先关键词：

1. 运动户外、户外、跑步、露营、骑行
2. 香氛、香水、美妆、个护、生活方式
3. 潮玩、IP、授权、动漫、游戏
4. 消费电子、智能硬件、AI、智能家居
5. 家居、家装、商业地产、购物中心、品牌快闪
6. 茶饮、咖啡、餐饮、连锁、加盟、零售
7. 宠物、母婴、健康食品

活动行数据需要标注：

1. 活动行页面URL
2. 报名/票价信息
3. 主办方
4. 参会人群或嘉宾
5. 活动类型：峰会、沙龙、闭门会、发布会、招商会、体验活动
6. 来源可信度：通常为中，高价值活动需补主办方官网或公众号复核

### 4.3 公众号/行业媒体

公众号是后续非常重要的数据源，原因是：

1. 很多峰会、沙龙、闭门会不会完整出现在展会聚合网站。
2. 公众号文章更容易包含嘉宾名单、代表品牌、议程、赞助商、参会岗位。
3. 行业媒体本身也是合作渠道和曝光渠道。

优先关注：

1. 中国建博会、CIFF中国家博会、AWE、CBE、ISPO、CCG EXPO 等官方号。
2. 灵兽传媒、销售与市场、泛家居圈、搜狐家居、新浪家居、美妆行业观察、香水时代、体育产业生态圈等行业媒体。
3. 主办方、协会、场馆、展馆、商业地产官方号。

公众号数据需要标注：

1. 公众号名称
2. 文章标题
3. 发布时间
4. 活动名称
5. 嘉宾/品牌/议程
6. 阅读/影响力线索
7. 是否适合作为商业曝光渠道

## 5. 头部活动判定建议

头部活动不只看规模，建议按以下维度打分：

1. 行业匹配度：是否属于 LHB A/B 类行业。
2. 品牌密度：是否能接触足够多潜在客户。
3. 决策层密度：是否出现CEO、创始人、市场负责人、渠道负责人。
4. 城市价值：是否在重点城市。
5. 来源可信度：官网/聚展网/主办方/公众号信息是否完整。
6. 内容价值：是否值得做专题页、榜单、城市观察或行业报告。
7. 商业转化：是否能承接赞助、线索表单、会刊购买、参展咨询。

## 6. 下一轮数据抓取建议

1. 聚展网按城市+行业关键词抓 50-100 条候选，补大展会、展览和展商规模。
2. 活动行按城市+行业关键词抓 50-100 条候选，补峰会、沙龙、闭门会、发布会、招商会和体验活动。
3. 只保留与 LHB 行业匹配的活动，剔除低相关工业展、纯供应链展、泛创业培训、泛职场课程。
4. 对每条头部活动补官网和公众号文章。
5. 对S/A级活动优先补图片、规模、代表品牌、参会岗位、门票、主办方。
6. 对B/C级活动允许先入库，但必须标注来源类型和可信度。
