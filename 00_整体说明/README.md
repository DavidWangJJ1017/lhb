# LHB 线下活动情报库项目说明

## 目录结构

| 文件夹 | 用途 |
|---|---|
| `00_整体说明` | 项目总说明、目录规则、后续维护说明 |
| `01_原始素材` | 用户提供的原始输入、外部原始参考资料、未加工素材 |
| `02_产出物` | PRD、数据分析、字段说明、生成报告、后续确认稿与交付文档 |
| `03_系统相关` | 网站页面、脚本、样式、前端数据文件、抓取缓存等系统运行文件 |

## 当前网站入口

网站文件位于：

`03_系统相关/website/index.html`

详情页入口：

`03_系统相关/website/detail.html`

如需本地预览，可在 `03_系统相关/website` 目录启动静态服务。

## 后续放置规则

1. 用户直接提供的 Excel、截图、外部参考报告、未加工资料，放入 `01_原始素材`。
2. 新增 PRD、分析报告、数据复盘、确认稿、生成型报告，放入 `02_产出物`。
3. 网站代码、数据 JS、样式、页面文件，放入 `03_系统相关/website`。
4. 抓取过程生成的 JSON、缓存、中间数据，放入 `03_系统相关/data-cache`。
5. 项目级说明、目录规则、维护备注，放入 `00_整体说明`。
